# X (Twitter) Recommendation Algorithm Simulator

A visual simulator for the X (Twitter) recommendation algorithm, based on the official open-source Heavy Ranker architecture and the latest 2026 weight adjustments.

🌍 **Live Demo:** [https://x-recommendation-simulator.vercel.app/](https://x-recommendation-simulator.vercel.app/)
🐦 **Author:** [@mac20777](https://x.com/mac20777)

## Features

- **Interactive Simulation:** Adjust parameters like initial followers, engagement rates (likes, retweets, replies, bookmarks), and see how they affect your tweet's reach.
- **Real-time Charts:** Visualize follower growth and engagement breakdown across multiple simulation cycles.
- **Algorithm Rulebook:** Learn about the core formula, in-network vs out-of-network barriers, penalties, and time decay.
- **Novice Guide:** Actionable Dos and Don'ts based on algorithmic insights.
- **Bilingual Support:** Switch seamlessly between English and Chinese.

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Recharts
- Framer Motion
- Lucide React
